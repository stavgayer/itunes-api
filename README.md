# Itunes-API-Ecercise

## Introduction

Itunes Video Searching exercise using : Node.js , React  and MongoDB.


##  How to install

```bash
git clone git@github.com:stavgayer/itunes-api.git
cd itunes-api

# Install backend dependencies
npm install

# Install frontend dependencies
cd client
npm install
```

## Running on dev mode 
```
./itunes-api

npm run dev
```
